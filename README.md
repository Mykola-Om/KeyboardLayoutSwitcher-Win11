# KeyboardLayoutSwitcher-Win11 (Перемикач розкладки)

Сучасна інтелектуальна утиліта для Windows 11, яка автоматично виправляє слова, що були помилково набрані в неправильній розкладці клавіатури (наприклад, між англійською та українською).

## 🌟 Основні можливості

- **Інтелектуальна автозаміна:** Автоматично розпізнає «абракадабру» (наприклад, "ghbdsn" -> "привіт") і перемикає розкладку, переписуючи слово правильно.
- **Нативний дизайн Windows 11:** Повноцінна підтримка спеціальної темної теми (Dark Mode) та сучасних елементів керування.
- **Гнучкий фільтр програм:** Вміє працювати в режимах «Включення» (працює виключно у вибраних програмах) або «Виключення» (працює всюди, крім обраних).
- **Сучасне управління списками:** Зручне додавання програм та слів-винятків у форматі сучасних "тегів" (чіпів). Програми можна додавати як вручну, так і через швидку кнопку огляду [ 📂 ].
- **Слова-винятки:** Можливість задати конкретні слова, які програма ніколи не буде перекладати.
- **Гнучке налаштування чутливості:** Ви самі контролюєте алгоритм розпізнавання (мінімальна довжина слова, відсоток співпадіння, різниця голосних). Біля кожного налаштування є підказки [ ⓘ ].
- **Робота у фоні:** Працює непомітно в системному треї (може запускатися разом із Windows).
- **Надійність:** Захист від подвійного запуску програми.

## 🚀 Як користуватись

1. Запустіть програму. Якщо вікно заважає, його можна сміливо закрити чи згорнути — програма продовжить роботу в системному треї.
2. Увімкніть **"Автоматична заміна розкладки"** (верхній перемикач).
3. За потреби налаштуйте: 
   - **програми**, де автозаміна має ігноруватися (наприклад, ігри чи термінал);
   - **слова-винятки**, які не слід чіпати алгоритму.
4. Пишіть текст природно! Якщо ви забули перемкнути мову, утиліта помітить це і автоматично виправить слово.

## ⚙️ Технічні деталі

Проєкт створено на C# (WinForms) з використанням бібліотек для глибокої інтеграції з нативними стилями ОС (DwmSetWindowAttribute, SetWindowTheme) та глобальними хуками клавіатури (SetWindowsHookEx). Це забезпечує мінімальне споживання ресурсів комп'ютера та миттєву швидкість реакції під час друку тексту. Всі налаштування зберігаються автоматично та застосовуються на льоту.
