# KeyboardLayoutSwitcher-Win11

Невелика утиліта для Windows, яка допомагає автоматично виправляти слова, набрані в неправильній розкладці клавіатури між українською та англійською.

## Що вміє

- автоматично замінювати слово на правильний варіант у потрібній розкладці;
- працювати у фоні через трей;
- вмикатися або вимикатися одним перемикачем;
- працювати тільки в обраних програмах або, навпаки, не працювати в них;
- дозволяти підлаштувати чутливість автозаміни;
- запускатися разом із Windows.

## Як користуватись

Запустіть програму, увімкніть автоматичну заміну та за потреби налаштуйте список програм, де вона має працювати. Якщо не хочете бачити вікно постійно, просто згорніть його в трей.

## Налаштування

У вікні програми можна:

- увімкнути або вимкнути автозаміну;
- додати автозапуск із Windows;
- вибрати режим фільтрації програм: без фільтра, білий список або чорний список;
- вказати назви процесів, для яких правило застосовується;
- змінити чутливість визначення неправильної розкладки.
